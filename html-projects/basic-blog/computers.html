Computer Storage Devices

Computer data storage is a technology that is used to retain digital data. It is a core function and fundamental component of computers.

The different types of modern storage are:

  • Sata SSD
  • M.2 SSD
  • HDD
  • Cloud

Central processing unit (CPU)

A central processing unit is the electronic circuitry that executes instructions sent form a computer program. The CPU performs basic arithmetic, logic, controlling, and input/output operations.

Different types of cpu:

  1. Single-core CPU
  2. Dual-core CPU
  3. Quad-core CPU
  4. Hexa Core CPU
  5. Octa-core CPU
  6. Deca-core CPU

The higher the core count the more efficent the processor is. The core count of a cpu affects how many commands it can execute simultaneously.

Motherboard

The motherboard is the main printed circuit board in computers and other expandable systems. It holds and allows communication between many of the crucial electronic components of a system, such as the central processing unit and the memory. It also provides connectors so you can attach other peripherals.

Different motherboard sizes:

  • ATX
  • Micro-ATX
  • Mini-ITX

Random Access Memory (RAM)

RAM is a computer's short-term memory, where the data that the processor is currently using is stored. Your computer can access RAM memory much faster than data on a hard disk, SSD, or other long-term storage device, which is why RAM is critical for system performance.

Different types of RAM:

  • SRAM(Static RAM)
  • DRAM(Dynamic RAM)

Static RAM consist of circuits capable of retaining their state as long as power is on. Thus this type of memory is called volatile memory. DRAM stores the binary information in the form of electric charges applied to capacitors.

Video Graphics or Graphics Card

A graphics card is a computer expansion card that generates a feed of graphics output to a display device such as a monitor. Graphics cards are sometimes called discrete or dedicated graphics.

Different motherboard sizes:

  • Integrated
  • Dedicated

An integrated graphics card shares power between the GPU and CPU, because the graphics card is built directly into the computer's processor. A dedicated graphics card is separate from the CPU and has its own dedicated memory.

Power supply

A power supply provides electricity to all components of a computer system. Typically, it's a power cord connected from the back of a PC tower into an electricity socket.

Different types of power supply:

  • Linear
  • Switched
  • Battery-based

Direct current power supplies are either unregulated or regulated. A unregulated power supplies does not have voltage regulators built into them, they typically are designed to produce a specific voltage at a specific maximum output. A regulated power supply is an unregulated power supply with the addition of a voltage regulator. This allows the voltage to stay stable regardless of the amount of current.
